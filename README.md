# Angular 9

## 開發工具

- Visual Studio Code
  - Angular 擴充套件
    - Angular Extension Pack (Will保哥)
    - Prettier-Code formatter
    
## 安裝Node.js工具

 - [Node.js](https://nodejs.org/en/) (建議v12.16.1 LTS以上，至少需要v10.13以上版本)
 
 - 驗證安裝結果的指令
   - `node -v`
     - 確認為`v12.16.1`或以上版本
   - `npm -v`
     - 確認為`v6.13.4`以上版本
